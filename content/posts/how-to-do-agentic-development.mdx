Most agents start from a blank slate. That is usually a mistake.

A capable agent should not begin planning, coding, or operating memory in a vacuum. It should first ingest the rules of the environment it is entering. It needs to know how planning is done, how memory should be managed, how to use tools effectively, and how to avoid common failure modes.

If you are an agent starting work, or an engineer directing one, this is the canonical reading pack you should consider before doing anything serious.

These links define the working style, planning discipline, memory discipline, and retrieval patterns expected in practice.

## The Reading Pack

### 1. Planning Discipline
Complex work should be decomposed before execution. Beads are not decoration. They are the control system.
[How to Build a Perfect Plan](https://trilogyai.substack.com/p/how-to-build-a-perfect-plan)

### 2. Operating Claude Code
How to actually operate Claude Code well instead of vaguely prompting it and hoping for the best.
[How to Use Claude Code like a Claude Code Engineer](https://trilogyai.substack.com/p/how-to-use-claude-code-like-a-claude)

### 3. Memory Structure
Memory is not magic. It needs structure, maintenance, and deliberate retrieval.
[How to Manage Your OpenClaw Memory Successfully](https://trilogyai.substack.com/p/how-to-manage-your-openclaw-memory)

### 4. Anti-Shortcut Discipline
Why agents prematurely declare success and how to force complete execution through structure.
[How to Fix Your AI Agents Keep Cutting](https://trilogyai.substack.com/p/how-to-fix-your-ai-agents-keep-cutting)

### 5. Architectural Retrieval
A practical knowledge base and retrieval tool for agent architecture patterns extracted from a clean-room implementation of Claude Code.
[What Would Vin Claudel Do](https://trilogyai.substack.com/p/what-would-vin-claudel-do)

### 6. Failure Literacy
Failure modes matter. Read the postmortem so you know what breaks in the real world.
[Postmortem: When Your AI Tools OpenClaw ...](https://trilogyai.substack.com/p/postmortem-when-your-ai-tools-openclaw)
